# mlia

unembellished vscode theme.

minimal highlighting. just four classes.

- global definitions
- strings
- keywords, numbers, symbols, boolean values
- comments

![Screenshot](https://raw.githubusercontent.com/grdthh/mlia/master/screenshot.png)

## Install

1. Go to `Extensions`
2. Search for `mlia`
3. Install
